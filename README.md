# Lab-5
Note App - приложение для заметок

Note App - это веб-приложение для создания и управления заметками. Позволяет создавать заметки с метками, изменять их статусы и фильтровать по меткам и статусам. Можно переводить заметки в архив, удалять их, также можно удалять метки. Backend на Python/Flask, фронтенд на чистом JavaScript, хранение в JSON.

#Sprint Review

Были закрыты задачи:
- Реализация CRUD API для заметок;
- Система меток;
- Пользовательский интерфейс

По итогам спринта было разработано приложение для создания и обновления заметок с возможностью добавления меток, а также приятный на вид интерфейс. Ведется работа по добавлению и изменению статусов заметки, после этого останется добавить фильтрацию по меткам и статусам - эти 2 задачи не успели завершить.

#Sprint Retrospective

Что прошло хорошо: четкое разделение ролей и ответственности - бэкендер успешно реализовал CRUD API и систему меток. Данные корректно сохраняются в JSON, все endpoint'ы работают. Frontend-разработчик создал чистый и визуально приятный интерфейс для взаимодействия с API. Пользоваться приложением стало интуитивно понятно. Задачи "CRUD API" и "Система меток" были выполнены полностью, создан работающий и цельный прототип приложения, который можно протестировать.

Что пошло не так: задачи "Система статусов" и "Фильтрация" оказались сложнее, чем предполагалось. Для статусов потребовалось переработать модель данных и обновить несколько API endpoint'ов. Фильтрация же зависит от готовности статусов. Код проверялся вручную, потому что задача "Настроить CI/CD pipeline" была вынесена из спринта из-за нехватки времени. Это отнимало время на тестирование и увеличивало риск пропустить ошибку.

Что можно улучшить в следующем спринте: перед следующим спринтом проводить короткие сессии совместной оценки сложности каждой задачи в story points. Обсуждать план предстоящей работы, чтобы выявить скрытые сложности. Обязательно включить CI/CD в следующий спринт для автоматической проверки. Учитывать зависимости при планировании, чтобы решать в первую очередь те задачи, от которых зависят другие.
